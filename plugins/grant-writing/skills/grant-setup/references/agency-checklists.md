# Default Document Checklists by Funder

Use these when the grants office has not supplied a checklist. Always reconcile against the specific FOA/NOFO — announcements add and remove documents. Mark the generated checklist "auto-generated; confirm with grants office."

## NIH (research grants, e.g., R01/R21/R03)

Verify against the FOA and current NIH Application Guide (grants.nih.gov). Typical items:

- [ ] Specific Aims (1 page)
- [ ] Research Strategy (R01: 12 pages; R21/R03: 6 pages) — Significance, Innovation, Approach
- [ ] Project Summary/Abstract (30 lines max) and Project Narrative (2–3 sentences)
- [ ] Biosketches, all senior/key personnel (5-page limit each, current NIH format)
- [ ] Other Support / current & pending (often Just-in-Time, but check FOA)
- [ ] Budget: modular (≤$250K direct/yr) or detailed (R&R form) + Budget Justification
- [ ] Facilities & Other Resources; Equipment
- [ ] Data Management and Sharing Plan (required since Jan 2023; 2-page recommended)
- [ ] Bibliography & References Cited
- [ ] Human Subjects: PHS Human Subjects and Clinical Trials Information form, inclusion plans
- [ ] Vertebrate Animals section (if applicable)
- [ ] Select Agents; Biohazards (if applicable)
- [ ] Authentication of Key Biological and/or Chemical Resources
- [ ] Letters of Support; Consortium/subaward documents (SOW, subaward budget)
- [ ] Multiple PI Leadership Plan (if multi-PI)
- [ ] Resource Sharing Plan(s) (model organisms, etc., if applicable)
- [ ] Cover Letter (optional; study section / institute assignment requests)
- Format: Arial/Georgia/Helvetica/Palatino Linotype ≥11 pt, ≥0.5" margins, PDF attachments

## NSF (standard research proposal per PAPPG 24-1 + supplements)

Verify against current PAPPG (NSF 24-1 as of mid-2026, plus Supplements 26-200/26-202) and the solicitation:

- [ ] Project Summary (1 page: Overview, Intellectual Merit, Broader Impacts)
- [ ] Project Description (15 pages default; must address Broader Impacts and Results from Prior NSF Support)
- [ ] References Cited
- [ ] Biographical Sketch(es) — SciENcv-generated, all senior personnel
- [ ] Current and Pending (Other) Support — SciENcv-generated
- [ ] Budget + Budget Justification (5-page limit)
- [ ] Facilities, Equipment and Other Resources
- [ ] Data Management and Sharing Plan (2 pages)
- [ ] Mentoring Plan (if postdocs funded; 1 page)
- [ ] Collaborators & Other Affiliations (COA) template per senior person
- [ ] Letters of Collaboration (format-restricted — no endorsement language) if allowed
- [ ] Synergistic activities, solicitation-specific supplements as required

## DoD (varies widely — CDMRP, DARPA, ONR, ARO, AFOSR)

Always start from the specific BAA/Program Announcement; DoD has no single format. Common pattern:

- [ ] Pre-application / white paper / letter of intent (often required first)
- [ ] Project Narrative (page limit per BAA)
- [ ] Statement of Work (SOW) — task/milestone structure, often a fixed template
- [ ] Budget and Justification (agency-specific forms)
- [ ] Biosketches/CVs per BAA format
- [ ] Quad chart (many CDMRP/DARPA mechanisms)
- [ ] Military relevance / impact statement (CDMRP)
- [ ] Data management plan, human/animal use documentation (service-specific approvals — HRPO/ACURO reviews add months post-award; flag early)
- [ ] Letters of support; regulatory documentation

## Foundations (generic)

Foundations vary most. Read the RFP; common items:

- [ ] Letter of Inquiry / pre-proposal (frequently the actual gate — treat with full rigor)
- [ ] Proposal narrative (often 3–10 pages, lay-accessible)
- [ ] Budget in foundation template; many cap or forbid indirect costs — flag for grants office
- [ ] PI biosketch/CV
- [ ] Institutional information, tax status letters (grants office provides)
- [ ] Letters of support
- [ ] Lay abstract / public summary (foundations weight these heavily)
